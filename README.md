# StoryCLM
### Технические особенности

Система StoryCLM это облачная платформа с возможностью горизонтального масштабирования, развернута в одном из самых больших и надежных датацентров в мире. Благодаря автоматическому масштабированию готова выдержать любую нагрузку на сервер. Кроме того, данные защищены на уровне сервера, а надежность обеспечивается ежедневным резервным копированием. Безопасность при передаче данных реализуется с помощью протокола https и сертификата безопасности на уровне домена (Geotrust True BusinessID Wildcard). 

### Установка

Чтобы развернуть систему, от клиента требуется список пользователей и установка приложения на свои планшеты. Для этого необходимо зайти с iPad на страницу http://storyclm.com и нажать кнопку “Установить”. Мобильное приложение совместимо с iPad 2 или новее, поддерживаются все iOS с версии 7.1 и новее.  

### Портал администрирования

Портал администрирования предоставляет все необходимые возможности для управления контентом. Важными техническими особенностями портала являются поддержка всех современных браузеров и адаптация для мобильных телефонов и планшетов. Загрузка и распределение контента между пользователями реализовано в несколько кликов. Имеется возможность редактировать контент и добавлять в презентацию любые вложения (pdf, видео) прямо на портале. Система предъявляет минимальные требования к верстке, программированию и файловой структуре презентации. Используя HTML5, контент может создать любое агентство или независимый разработчик. При этом существует мощное системное API расширяющее возможности разработчиков для создания интерактивных приложений (презентаций) любой сложности. Для целевого общения с пользователями планшетов на портале существуют 3 канала: email-уведомления, push-уведомления и HelpDesk с чатом внутри iOS приложения.

### CLM система 2.0 (CLM система нового поколения)

StoryCLM позволяет связать множество внешних источников данных воедино,  замкнув на себе огромные потоки информации. Способность системы интегрироваться с ERP и CRM системами, сторонними приложениями и устройствами по сбору данных дает возможность агрегировать информацию с внешних источников, накапливать и анализировать большие массивы данных. Большинство операций по обработке данных происходят в реальном времени по мере их поступления и пользователь может мгновенно увидеть результаты в виде аналитических графиков и отчетов.

Система использует принцип асинхронного обмена данными. Это позволяет удобно работать с презентациями как online так и offline. В независимости от наличия интернета данные полученные в результате работы при появлении доступа в сеть в полном объеме и без ошибок будут загружены на сервер и, при необходимости, будут доставлены во внешние системы. И наоборот, данные из StoryCLM и внешних систем будут загружены на планшет. При таком подходе, даже временная недоступность системы или внешних систем не повлечет за собой даже частичную потерю информации. Данные выстроятся в очередь и будут ожидать пока внешняя система будет готова их принять.

Презентации в StoryCLM, благодаря уникальному API, способны хранить и обмениваться между собой и внешними системами данными любой структуры. В рамках системы, у презентаций появляется собственная база данных, что позволяет ей собирать, оперировать и хранить на сервере  массивы информации. В совокупности с возможностью обмениваться данными с другими презентациями, сервером или внешними системами (ERP, CRM),  появляется возможность выйти за рамки обычных презентаций.  StoryCLM дает возможность создавать полноценные бизнес приложения.
