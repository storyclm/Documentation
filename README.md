# StoryCLM

[StoryCLM](http://breffi.ru/storyclm) — разработанная [BREFFI](http://breffi.ru/) диджитал-платформа, позволяющая создавать интерактивные презентации (приложения) c возможностью получения моментальной обратной связи как об изменении восприятия бренда клиентом, так и о работе самого представителя.

## Технические особенности
Система StoryCLM — это облачная платформа с возможностью горизонтального масштабирования.

Она легко интегрируется с ERP- и CRM-системами, сторонними приложениями и устройствами по сбору данных. В основе используется принцип асинхронного обмена данными.

Данные, накопленные всеми участниками системы (мобильными приложениями и внешними источниками), гарантированно и без ошибок загружаются на сервер и доставляются адресатам в независимости от доступности интернет-подключения.

### Установка

Чтобы развернуть систему, от клиента требуется список пользователей и установка приложения на свои планшеты. Для этого необходимо зайти с iPad на страницу http://storyclm.com и нажать кнопку “Установить”. Мобильное приложение совместимо с iPad 2 или новее, поддерживаются все iOS с версии 7.1 и новее.
